# RayTracer in C++
Extended version of ray tracer based on Peter Shirley's "Ray Tracer in One Weekend"

## Basic Content

* Basic ray casting
* Positionable camera
* Sphere intersections
* Antialiasing
* Lambertian Surfaces
* Reflection
* Refraction
* Depth of Field

## Added Content

* Cubes made of triangles
* Normal shadows and soft shadows
* Point lights 
* Directional lights 
* Blinn Materials
